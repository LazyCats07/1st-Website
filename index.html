Pengertian

HTML adalah singkatan dari Hypertext Markup Language. Pengertian HTML sering kali dikaitkan dengan bahasa pemrograman. Akan tetapi, sejatinya, HTML bukan sebuah bahasa pemrograman (programming languange), melainkan sebuah markup languange. uwu Apakah kamu semakin bingung dengan penjelasan itu? Hypertext Markup Language atau HTML adalah bahasa markup yang digunakan untuk membuat sebuah halaman web. Isinya terdiri dari berbagai kode yang dapat menyusun struktur suatu website. lol

HTML terdiri dari kombinasi teks dan simbol yang disimpan dalam sebuah file. Dalam membuat file HTML, terdapat standar atau format khusus yang harus diikuti. Format tersebut telah tertuang dalam standar kode internasional atau ASCII (American Standard Code for Information Interchange).

Contoh Code HTML

    <html>
    <head>
	 <title>Welcome to campuslife.</title>
    </head>
    <body>
	<h1>Campuslife Wesbite</h1>
	<h2>My first page.</h2>
	<p>Campuslife is a free learning source website for users.<p>  
	</body>
    </html>

Tempat Belajar HTML

Belajar HTML GRATIS untuk Pemula :

Anda bisa belajar HTML secara online, dan yang paling penting adalah gratis. Berikut ini adalah beberapa website atau situs di mana Anda bisa belajar bahasa markah HTML secara online.

1. Dicoding

Saat ini di dicoding, anda dapat mendapatkan kesempatan untuk belajar programing di bidng front end.

Kesempatan tersebut jika kalian mendaftar pada website baparekraf atau kementrian yang menyediakan beasiswa atau kesempatan belajar gratis untuk belajar coding.

Tidak hanya unsur html saja yang ada di dalam paket belajar tersebut, tetapi ada pula scc dan Java script yang dapat anda pelajari di web berikut

2. Coursera

Coursera adalah platform untuk belajar apapun termasuk mengenai koding. Nah, belajar di platform ini memungkinkan Anda untuk belajar dari pengajar yang kompeten.

Yang jelas, di sini Anda bisa belajar mengenai HTML dan juga CSS. Selain itu, Anda juga bisa belajar hal lain seperti Python, Java, bahasa pemrograman C dan lain-lain.

Jika Anda berniat untuk mencetak sertifikat karena menyelesaikan kursus di Coursera, maka harus ada sejumlah nominal yang harus dibayarkan.

3. Sololearn

Sololearn mengajarkan Anda untuk membuat kode dalam format yang menyenangkan, gratis, dan berukuran kecil. Anda akan menemukan lebih dari 20 bahasa pengkodean, termasuk Python, JavaScript, HTML, SQL, dan C++. Setiap kursus dirancang oleh programmer ahli dan disesuaikan dengan level Anda.
